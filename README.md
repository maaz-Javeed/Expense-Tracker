# Personal Expense Tracker

## Description
A Java Swing application for tracking personal income and expenses.

## Features
- Add income
- Add expenses
- Delete transactions
- Save data to file
- Load data from file
- Calculate balance

## Technologies
- Java
- Swing
- JUnit 4
- Git & GitHub

## How to Run
1. Open project in IntelliJ IDEA.
2. Run ExpenseTrackerGUI.java.
3. Use the interface to manage transactions.
